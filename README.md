## 🚆 Task 1 – Online Railway Reservation System

A console-based railway reservation system developed using **Java, JDBC, and MySQL**.

### Features

- Secure User Login
- Train Search
- Ticket Reservation
- Automatic PNR Generation
- Ticket Cancellation
- Database Integration
- Input Validation
- Booking Management
